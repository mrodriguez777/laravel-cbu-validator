# Laravel Cbu Validator
CBU Validator for Laravel 5

## Install

Via Composer

``` bash
$ composer require mrodriguez777/laravel-cbu-validator
```

Set providers in app/config.php
``` php
'providers' => [
    Mrodriguez777\CbuValidator\CbuValidatorServiceProvider::class,
]
```
